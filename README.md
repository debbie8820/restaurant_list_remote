#h1 Top Restaurants in Taiwan
這是個為想在台北市尋找美食的人所設計的簡單網站，讓大家輕鬆找到美食落角處。

#h3 功能
+ 快速瀏覽餐廳的概括資訊
+ 關鍵字搜尋餐廳
+ 點選餐廳圖片得知更多細節
+ 點擊導覽列左上角的"Top Restaurants"回到首頁

#h3環境建置與需求
+ Node.js: v10.15.0

#h3 安裝與執行
1. 打開終端機將專案下載至本地執行
`git clone https://github.com/debbie8820/restaurant_list_remote.git`

2. 進入專案資料夾
`cd restaurant_list`

3. 安裝專案需求
`npm install`

4. 啟動伺服器並執行專案
`npm run dev`
當終端機顯示`ready!`時即可到http://localhost:3000查看專案
